Soy Dilson Zuleta, un profesional que opera en la intersección entre análisis de datos, desarrollo de software y automatización de procesos de negocio.

Mi enfoque no se limita a escribir código o construir reportes. Mi trabajo comienza entendiendo el problema de negocio, analizando los datos disponibles, diseñando la solución técnica más adecuada y ejecutándola con precisión — ya sea un dashboard ejecutivo, una aplicación interna, un flujo automatizado o una integración entre sistemas.

Combino pensamiento analítico con capacidad de desarrollo para crear soluciones que no solo funcionan, sino que transforman la forma en que las organizaciones operan, deciden y crecen.
